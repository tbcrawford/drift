<action>
**Step 1:** From repo root run:

```bash
go get github.com/spf13/cobra@v1.9.1
go mod tidy
```

**Step 2:** Create `cmd/drift/main.go` with `package main` containing:

- `rootCmd := &cobra.Command{ Use: "drift [flags] OLD NEW", Short: "Pretty-print a diff between two inputs", SilenceUsage: true, SilenceErrors: true, Args: cobra.MaximumNArgs(2), RunE: func(cmd *cobra.Command, args []string) error { return fmt.Errorf("not implemented: complete in plan 05-02") } }`
- Register flags using exact long names:
  - `rootCmd.Flags().Bool("split", false, "side-by-side split view")`
  - `rootCmd.Flags().String("algorithm", "myers", "diff algorithm: myers, patience, histogram")`
  - `rootCmd.Flags().String("lang", "", "Chroma language override (e.g. go, python)")`
  - `rootCmd.Flags().String("theme", "", "Chroma style/theme override")`
  - `rootCmd.Flags().Bool("no-color", false, "disable ANSI colors")`
  - `rootCmd.Flags().Int("context", 3, "lines of context around hunks")`
  - `rootCmd.Flags().String("from", "", "old text as a raw string (use with --to)")`
  - `rootCmd.Flags().String("to", "", "new text as a raw string (use with --from)")`

**Step 3:** `main()` calls `if err := rootCmd.Execute(); err != nil { fmt.Fprintln(os.Stderr, err); os.Exit(2) }` (exit code 2 for runtime errors before diff wiring is acceptable in this intermediate plan).

**Step 4:** Add `cmd/drift/main_test.go` with a test that executes `rootCmd.SetArgs([]string{"--help"})`, captures combined output to a buffer, and asserts the output substring contains each of: `--split`, `--algorithm`, `--lang`, `--theme`, `--no-color`, `--context`, `--from`, `--to`.
</action>
