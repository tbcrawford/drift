    ```go
    package myers

    import "github.com/tbcrawford/drift"

    type Myers struct{}

    func New() *Myers { return &Myers{} }

    func (m *Myers) Diff(oldLines, newLines []string) []drift.Edit {
        // Handle edge cases first
        if len(oldLines) == 0 && len(newLines) == 0 {
            return []drift.Edit{}
        }
        // ... Myers SES algorithm
    }
    ```

    Algorithm structure:
    1. Handle edge cases (both empty, one empty)
    2. Pre-allocate V array of size 2*(N+M)+1 (avoid repeated allocation)
    3. Forward pass: iterate d from 0 to N+M; for each d iterate k; extend each diagonal; detect endpoint
    4. **CRITICAL**: save copy of V into trace slice at END of d-loop (after the k-loop)
    5. Backtrack: from (N,M) backwards through trace, emit Equal/Delete/Insert edits
    6. Reverse edit slice (backtrack produces reverse order)

    Ensure `Myers` implements `algo.Differ` — add a compile-time check:
    ```go
    var _ algo.Differ = (*Myers)(nil)
    ```
