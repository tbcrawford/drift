      → 2 hunks; first ends at line 5; second starts at line 16

    - Test 5: Two nearby changes → single merged hunk (overlap context)
      old = 10 lines; changes at line 3 and line 7; context=3
      → 1 merged hunk (context windows overlap: line 3+3=6, line 7-3=4 → merge)

    - Test 6: WithContext(0) → no context lines, only changed lines
      → hunk contains only Insert/Delete lines, no Equal lines
  </behavior>
  <action>
    Implement `hunk.Build(edits []drift.Edit, oldLines, newLines []string, contextLines int) []drift.Hunk`:

    Algorithm:
    1. Collect indices of all non-Equal edits
    2. Group into ranges: [start-contextLines, end+contextLines], clamped to [0, len-1]
    3. Merge overlapping ranges: if range[i].end >= range[i+1].start - 1, merge
    4. For each merged range, walk through the edits in that line range:
       - Build []Line from oldLines/newLines using the Edit sequence
       - Equal edits → Line{Op: Equal, Content: oldLines[i], OldNum: edit.OldLine, NewNum: edit.NewLine}
       - Delete edits → Line{Op: Delete, Content: oldLines[edit.OldLine-1], OldNum: edit.OldLine}
       - Insert edits → Line{Op: Insert, Content: newLines[edit.NewLine-1], NewNum: edit.NewLine}
    5. Compute hunk header fields from the Lines slice:
       - OldStart = first line's OldNum (skip inserts to find first)
       - OldLines = count of Equal + Delete lines
       - NewStart = first line's NewNum (skip deletes to find first)
       - NewLines = count of Equal + Insert lines
  </action>

    ```go
    package drift

    import (
        "strings"

        "github.com/tbcrawford/drift/internal/algo/myers"
        "github.com/tbcrawford/drift/internal/hunk"
    )

    // Diff computes the line-level diff between old and new using the configured
    // algorithm and returns a structured DiffResult.
    //
    // Line endings are normalized: Windows \r\n is treated as \n.
    // When both inputs are identical, DiffResult.IsEqual is true and Hunks is empty.
    func Diff(old, new string, opts ...Option) (DiffResult, error) {
        cfg := defaultConfig()
        for _, o := range opts {
            o(cfg)
        }

        // Normalize line endings: \r\n → \n
        old = strings.ReplaceAll(old, "\r\n", "\n")
        new = strings.ReplaceAll(new, "\r\n", "\n")

        // Split into lines, trimming trailing empty string from final newline
        oldLines := splitLines(old)
        newLines := splitLines(new)

        // Fast path: identical inputs
        if old == new {
            return DiffResult{IsEqual: true}, nil
        }

        // Dispatch to algorithm
        var differ algoInterface
        switch cfg.algorithm {
        case Patience, Histogram:
            // Patience and Histogram are implemented in Phase 2.
            // Fall back to Myers for now (they will replace this in Phase 2).
            differ = myers.New()
        default: // Myers
            differ = myers.New()
        }

        edits := differ.Diff(oldLines, newLines)
        hunks := hunk.Build(edits, oldLines, newLines, cfg.contextLines)

        return DiffResult{
            Hunks:   hunks,
            IsEqual: len(hunks) == 0,
        }, nil
    }

    // splitLines splits text by \n and removes the trailing empty element
    // produced when the text ends with a newline.
    func splitLines(text string) []string {
        lines := strings.Split(text, "\n")
        if len(lines) > 0 && lines[len(lines)-1] == "" {
            lines = lines[:len(lines)-1]
        }
        return lines
    }

    // algoInterface is the internal interface for algorithm implementations.
    // Matches internal/algo.Differ but defined here to avoid import cycle.
    type algoInterface interface {
        Diff(oldLines, newLines []string) []Edit
    }
    ```

    Note: The `algoInterface` local definition avoids importing `internal/algo` in the root package (which already defines the `Edit` type that `internal/algo` imports). This avoids an import cycle. The `internal/algo/myers` package can import root for the `Edit` type; root doesn't need to import `internal/algo`.
