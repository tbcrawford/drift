# Plan 02-02: Implement Histogram diff in `internal/algo/histogram/`

## Objective
Implement Git's Histogram diff algorithm as an `algo.Differ`-satisfying package. The algorithm iteratively finds the longest contiguous matching block with the lowest occurrence count in A, then recurses on before/after regions, falling back to Myers when all A lines exceed `histogramMaxOccurrences = 64`.

**func (h *Histogram) Diff(old, new []string) []edittype.Edit** — iterative stack implementation:

1. Handle edge cases:
   - Both empty → return `[]edittype.Edit{}`
   - old empty → return M Insert edits with NewLine=i+1, OldLine=0
   - new empty → return N Delete edits with OldLine=i+1, NewLine=0

2. Initialize: `stack := []frame{{0, len(old), 0, len(new)}}`, `edits := make([]edittype.Edit, 0, len(old)+len(new))`

3. Main loop `for len(stack) > 0`: pop from end.

4. Per frame `f`:
   a. Skip if `f.os == f.oe && f.ns == f.ne` → continue
   b. Old range empty: emit `f.ne-f.ns` Insert edits with `OldLine=0, NewLine=f.ns+i+1` → continue
   c. New range empty: emit `f.oe-f.os` Delete edits with `OldLine=f.os+i+1, NewLine=0` → continue
   d. Build occurrence map: `counts := make(map[string]int)` for `old[f.os:f.oe]`
   e. Call `findBestMatch(old, new, f, counts)` → returns `(matchResult, bool)`
   f. If no match found (`!found`): call Myers fallback `myers.New().Diff(old[f.os:f.oe], new[f.ns:f.ne])`, apply offsets (`OldLine += f.os` when > 0, `NewLine += f.ns` when > 0), append to edits, continue
   g. If match found:
      - Push `frame{match.oe, f.oe, match.ne, f.ne}` (after-match region) if non-empty
      - Push `frame{f.os, match.os, f.ns, match.ns}` (before-match region) if non-empty
      - Emit `match.oe - match.os` Equal edits: `edittype.Edit{Op: edittype.Equal, OldLine: match.os+i+1, NewLine: match.ns+i+1}` for i in 0..len

**Push order for stack:** Push after-match first, then before-match. Since we pop from the end, before-match is processed first (file order maintained).

**func findBestMatch(old, new []string, f frame, counts map[string]int) (matchResult, bool)**:

Implementation follows jgit HistogramDiff pseudocode from research section 1.2:

```
lowcount := histogramMaxOccurrences + 1  // start above threshold
var best matchResult
found := false

for ni := f.ns; ni < f.ne; ni++ {
    line := new[ni]
    cnt, ok := counts[line]
    if !ok || cnt > lowcount { continue }  // skip if not in old or above current best; cnt is the B-side (new) line's occurrence count in old
    
    // Find all positions in old[f.os:f.oe] where this line occurs
    for oi := f.os; oi < f.oe; oi++ {
        if old[oi] != line { continue }
        if counts[line] > lowcount { continue }  // A-side check: skip if this old position's line exceeds current best (both outer B-side and inner A-side checks are intentional)
        
        // Widen match forward and backward within frame bounds
        matchOs, matchNs := oi, ni
        matchOe, matchNe := oi+1, ni+1
        
        // Widen backward
        for matchOs > f.os && matchNs > f.ns && old[matchOs-1] == new[matchNs-1] {
            matchOs--; matchNs--
        }
        // Widen forward
        for matchOe < f.oe && matchNe < f.ne && old[matchOe] == new[matchNe] {
            matchOe++; matchNe++
        }
        
        // Compute lowest occurrence count in matched old range
        regionLow := histogramMaxOccurrences + 1
        for k := matchOs; k < matchOe; k++ {
            if c := counts[old[k]]; c < regionLow { regionLow = c }
        }
        
        matchLen := matchOe - matchOs
        bestLen := best.oe - best.os
        if !found || matchLen > bestLen || (matchLen == bestLen && regionLow < lowcount) {
            best = matchResult{matchOs, matchOe, matchNs, matchNe}
            lowcount = regionLow
            found = true
        }
    }
}
return best, found
```
