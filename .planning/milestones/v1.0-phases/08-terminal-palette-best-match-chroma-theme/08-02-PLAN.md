## Objective

When **no explicit theme** is set, **TTY + color profile allows**, and **Unix**: query OSC 4 via `/dev/tty`, parse palette, set Chroma style via `BestMatchTheme`. On failure, timeout, non-Unix, or non-file writer: **existing** `SelectTheme("", isDark)` path. Add optional **`--show-theme`** stderr line. Document behavior in README.

## must_haves

- [ ] Explicit `--theme` / `WithTheme` bypasses OSC 4 entirely
- [ ] Piped stdout / `NO_COLOR` / non-TTY: no OSC 4 attempt (no hang)
- [ ] `go test ./... -count=1` exits 0
