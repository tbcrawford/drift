Rewrite `runRoot` in `cmd/drift/main.go` to:

1. Resolve inputs (same as before).
2. Call `drift.Diff(...)` (same as before).
3. If `result.IsEqual` → return nil (same as before).
4. Render to a `bytes.Buffer` (not directly to `streams.Out`).
5. Count lines in the buffer: `lineCount := strings.Count(buf.String(), "\n")`.
6. Get terminal height: if `streams.Out` is `*os.File`, call `term.GetSize(f.Fd())` to get height; else height=0.
7. Call `shouldPage(streams.Out, lineCount, height, opts.noPager)`.
8. If shouldPage returns true: call `startPager(resolvePager(), streams)` → write buffer to pagerWriter → call cleanup.
9. If shouldPage returns false: write buffer directly to `streams.Out`.
10. Return `newExitCode(1, "")`.

The render call must use `drift.RenderWithNames(result, &buf, oldName, newName, opts.driftOpts...)` (render to buffer, not streams.Out directly).

```go
func TestRunCLI_noPagerFlag(t *testing.T) {
    var out, errOut bytes.Buffer
    streams := IOStreams{In: strings.NewReader(""), Out: &out, Err: &errOut}
    code := runCLI(streams, []string{"--from", "a\n", "--to", "b\n", "--no-pager"})
    if code != 1 {
        t.Fatalf("expected exit 1, got %d stderr=%q", code, errOut.String())
    }
    if !strings.Contains(out.String(), "@@") {
        t.Fatalf("expected diff hunk header in output: %q", out.String())
    }
}
```
