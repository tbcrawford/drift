<objective>
Two targeted optimizations on top of the Plan 01 wt.Status() fix:

1. **Streaming render to pager** — currently `runGitDirectoryDiff` renders ALL files into
   `bytes.Buffer` before the pager sees any output. For 25 files, this means ~200ms of
   hidden latency before the terminal updates. Switch to streaming: render each file's
   diff into a temp buffer, write it to the pager incrementally. User sees the first
   file's diff within milliseconds.

2. **Eliminate redundant repo opens** — `gitDirectoryVsHEAD` calls `openRepoAt` once,
   then each `gitShowHEADBlob` call inside the loop calls `openRepoAt` again. For 25
   files = 26 `PlainOpenWithOptions` calls. Refactor to open once and pass the
   `*git.Repository` + `*object.Tree` to all callers.

   Measured cost of each extra open: ~200µs × 25 = 5ms. Small but eliminates unnecessary
   syscalls and the pattern is wrong.

Purpose: Push from sub-500ms (plan 01 target) toward sub-200ms for typical repos.
Output: `streamThroughPager`, refactored `gitShowHEADBlob(repo, tree, path)` signature.
</objective>

**Problem:** `runRoot` (zero-arg git mode) buffers all 25 files into `bytes.Buffer`,
then starts the pager. User waits ~300ms before seeing anything.

**Fix:** For the git-directory diff path, start the pager immediately (when stdout is a TTY),
then stream rendered file diffs directly into the pager writer as they're computed.

Add `streamThroughPager` alongside `writeThroughPager`:

```go
// streamThroughPager starts the pager immediately (if TTY), then calls renderFn
// which writes rendered diff content directly to the pager/stdout writer.
// Rendering happens inside renderFn — no bytes.Buffer needed.
//
// If the pager cannot be started, falls back to writing to opts.streams.Out directly.
// Returns newExitCode(1, "") on success (diff found), nil on no diff.
func streamThroughPager(opts *rootOptions, renderFn func(w io.Writer) (hasDiff bool, err error)) error {
    if _, ok := opts.streams.Out.(*os.File); ok && !opts.noPager {
        // TTY path: start pager before any rendering.
        pagerWriter, cleanup, pErr := startPager(resolvePager(), opts.streams)
        if pErr == nil {
            hasDiff, renderErr := renderFn(pagerWriter)
            cleanup()
            if renderErr != nil && !isPipeClosedErr(renderErr) {
                return newExitCode(2, renderErr.Error())
            }
            if !hasDiff {
                return nil
            }
            return newExitCode(1, "")
        }
        // Pager failed to start — fall through to direct write.
    }
    // Non-TTY or pager fallback: write directly to stdout.
    hasDiff, err := renderFn(opts.streams.Out)
    if err != nil {
        return newExitCode(2, err.Error())
    }
    if !hasDiff {
        return nil
    }
    return newExitCode(1, "")
}
```

**Update `runGitDirectoryDiff`** to accept an `io.Writer` instead of `*bytes.Buffer`:

```go
// runGitDirectoryDiff renders git HEAD-vs-working-tree diffs for all changed files
// into w, with styled file headers. Diffs computed in parallel; output written in
// sorted order. Returns hasDiff=true if any file pair produced output.
func runGitDirectoryDiff(pairs []gitFilePair, opts *rootOptions, w io.Writer) (hasDiff bool, err error) {
    if len(pairs) == 0 {
        return false, nil
    }

    results := make([]bytes.Buffer, len(pairs))
    g := new(errgroup.Group)

    for i, pair := range pairs {
        i, pair := i, pair
        g.Go(func() error {
            result, dErr := drift.Diff(pair.HeadContent, pair.WorkContent, opts.driftOpts...)
            if dErr != nil {
                return newExitCode(2, dErr.Error())
            }
            if result.IsEqual {
                return nil
            }
            writeFileHeader(&results[i], pair.Name, opts.noColor, opts.termWidth)
            return drift.RenderWithNames(result, &results[i], pair.OldName, pair.NewName, opts.driftOpts...)
        })
    }

    if err := g.Wait(); err != nil {
        return false, err
    }

    for i := range results {
        if results[i].Len() > 0 {
            hasDiff = true
            if _, wErr := results[i].WriteTo(w); wErr != nil {
                return hasDiff, wErr
            }
        }
    }
    return hasDiff, nil
}
```
