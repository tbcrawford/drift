## Objective

Introduce **shared** gutter formatting (column width scan, centered ASCII numerals, blank padding for `0` line numbers, Lip Gloss gutter styles with `NoColor` fallback) and wire **unified** diff output so each line shows **old | new** gutters before the `+`/`-`/space prefix and highlighted code. Add **`drift.WithLineNumbers`** (default `true`) and plumb **`ShowLineNumbers`** through `render.RenderConfig` in `render.go`.

## must_haves

- [ ] `go test ./internal/render/... -count=1` exits 0
- [ ] `go test ./... -run TestRender -count=1` exits 0 (or full `./...` if fast)
- [ ] Unified output with line numbers disabled matches prior behavior (no regression) — optional `WithoutLineNumbers()` or `WithLineNumbers(false)`

- Concatenate: `leftGutterRendered + sep + rightGutterRendered + prefix + highlighted + "\n"`.

**Step 5:** Pass **isDark** into gutter style: add `IsDark bool` to `RenderConfig` OR call an existing helper — `render` must not import `internal/theme` if it creates cycles; **prefer** passing `isDark bool` from `render.go` (`theme.DetectDarkBackground` already computed there as `isDark`).

**Step 6:** Add `IsDark bool` field to `RenderConfig` set in `render.go` from existing `isDark` variable.
