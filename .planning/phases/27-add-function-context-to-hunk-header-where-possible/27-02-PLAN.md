<objective>
Thread git's `code_fragment` (captured in Plan 01's `parsedFileDiff.Hunks`) through
the rendering pipeline so it appears in `@@ -x,y +a,b @@ func_name` hunk headers.

Purpose: This is the visible output change. Plan 01 captured the data; this plan
displays it. The approach: add `CodeFragment string` to `edittype.Hunk`
(populated only in the git pager path, never in standalone `drift.Diff()` mode),
then update unified/split renderers to append it.

**Key constraint (D-02/D-08 from CONTEXT.md):** The standalone `drift.Diff()` path
is UNCHANGED. `CodeFragment` is populated exclusively by `runPagerMode` by
backfilling result.Hunks after the diff is computed, matching by position.
No `WithFuncContext()` option, no regex, no language detection.

Output: `edittype.Hunk` gains `CodeFragment string`; renderers use it; `runPagerMode`
backfills it from `parsedFileDiff.Hunks`; integration test verifies end-to-end.
</objective>

CodeFragment backfill step before rendering:

```go
result, dErr := drift.Diff(f.OldContent, f.NewContent, opts.driftOpts...)

    return false, newExitCode(2, dErr.Error())

if result.IsEqual {
    continue
}

// Backfill CodeFragment from git @@ headers into each rendered hunk.
// Drift re-computes hunks independently; we match them by position.
// If counts differ (edge case), we fill as many as we can.

    if i < len(f.Hunks) {
        result.Hunks[i].CodeFragment = f.Hunks[i].CodeFragment

}
```

This goes immediately after the `result.IsEqual` check, before the `bytes.Buffer` is written.
The import for `drift` is already present in `cmd/drift/main.go`.

(no interface boxing). Verify `result.Hunks` is a `[]edittype.Hunk` value slice — modifying

```go
// TestRunPagerMode_CodeFragment verifies that when drift acts as a git pager,
// the code_fragment from git @@ headers appears in the rendered @@ hunk header.
func TestRunPagerMode_CodeFragment(t *testing.T) {
    // Simulate git diff output with a code_fragment on the @@ line.
    gitDiff := `diff --git a/options.go b/options.go
--- a/options.go
+++ b/options.go
@@ -5,6 +5,6 @@ func ParseOptions(args []string)
 package main
 
 // ParseOptions parses command-line arguments.
-func ParseOptions(args []string) error {
+func ParseOptions(args []string) (Options, error) {
 	return nil
 }
`
    files, err := parseUnifiedDiff(strings.NewReader(gitDiff))
    if err != nil {
        t.Fatalf("parseUnifiedDiff: %v", err)
    }
    if len(files) != 1 {
        t.Fatalf("expected 1 file, got %d", len(files))
    }
    f := files[0]
    if len(f.Hunks) != 1 {
        t.Fatalf("expected 1 hunk fragment, got %d", len(f.Hunks))
    }
    if f.Hunks[0].CodeFragment != "func ParseOptions(args []string)" {
        t.Errorf("CodeFragment = %q, want %q",
            f.Hunks[0].CodeFragment, "func ParseOptions(args []string)")
    }
    // Verify that the code_fragment survives until render time by exercising
    // the backfill logic directly (simulate what runPagerMode does).
    import_drift "github.com/tbcrawford/drift"
    result, err := import_drift.Diff(f.OldContent, f.NewContent)
    if err != nil {
        t.Fatalf("drift.Diff: %v", err)
    }
    for i := range result.Hunks {
        if i < len(f.Hunks) {
            result.Hunks[i].CodeFragment = f.Hunks[i].CodeFragment
        }
    }
    if len(result.Hunks) < 1 {
        t.Fatal("expected at least one hunk in diff result")
    }
    if result.Hunks[0].CodeFragment != "func ParseOptions(args []string)" {
        t.Errorf("after backfill: Hunks[0].CodeFragment = %q, want %q",
            result.Hunks[0].CodeFragment, "func ParseOptions(args []string)")
    }

    // Render and verify @@ header contains the code_fragment.
    var buf strings.Builder
    renderErr := import_drift.Render(result, &buf, import_drift.WithNoColor())
    if renderErr != nil {
        t.Fatalf("Render: %v", renderErr)
    }
    output := buf.String()
    if !strings.Contains(output, "@@ ") {
        t.Error("rendered output does not contain @@ hunk header")
    }
    if !strings.Contains(output, "func ParseOptions") {
        t.Errorf("rendered output does not contain code_fragment 'func ParseOptions':\n%s", output)
    }
}
```
