```
// filePair represents a single file entry in a directory diff.
type filePair struct {
    Name    string // relative path with forward slashes (display name)
    OldPath string // absolute path on old side; empty if file is added
    NewPath string // absolute path on new side; empty if file is removed
}

// IsAdded reports whether the file exists only on the new side.
func (fp filePair) IsAdded() bool { return fp.OldPath == "" }

// IsRemoved reports whether the file exists only on the old side.
func (fp filePair) IsRemoved() bool { return fp.NewPath == "" }
```

`diffDirectories(oldDir, newDir string) ([]filePair, error)`:
1. `os.Stat` both dirs; return error if either is not a directory.
2. Walk `oldDir` with `filepath.WalkDir`; for each regular file build a map[relName → absOldPath].
3. Walk `newDir` with `filepath.WalkDir`; for each regular file:
   - If it also exists in old map: read both files; if content differs → add filePair with both paths.
   - If it does not exist in old map → add filePair with empty OldPath (IsAdded).
   - Delete from old map after processing.
4. For each remaining key in old map → add filePair with empty NewPath (IsRemoved).
5. Sort result slice by Name using `slices.SortFunc`.
6. Return sorted slice.

Use `filepath.ToSlash(rel)` for the display Name so Windows paths use forward slashes.
Use `os.ReadFile` to read file contents for comparison (byte equality check — no diff needed here).
Do NOT use `filepath.Walk`; use `filepath.WalkDir` (more efficient, available since Go 1.16).
Symlinks are skipped (only `DirEntry.Type().IsRegular()` files are collected).
