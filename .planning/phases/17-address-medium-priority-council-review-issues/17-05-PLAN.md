**Step C (if compatible): Migrate palette_unix.go**

In `internal/terminal/palette_unix.go`, change:
```go

oldState, err := term.MakeRaw(int(tty.Fd()))
// ...
```

To:
```go
import "github.com/charmbracelet/x/term"

```

Note: `charmbracelet/x/term.State` and `golang.org/x/term.State` are different
types. The variable `oldState` type is inferred — no explicit type annotation
needed, just update the import and remove the `int()` casts.
