## Objective

Implement input resolution: two file paths (`drift a b`), stdin placeholder `-` paired with a file (`drift - b` / `drift a -`), raw strings (`--from` + `--to`), and `drift - -` (single `io.ReadAll(os.Stdin)` reused for both sides â€” identical inputs, empty diff). Validate mutual exclusivity: either exactly two positional operands **or** both `--from` and `--to` set (non-empty). Return typed errors for misuse (message must include `usage` or `invalid` for test matching).

## must_haves

- [ ] `readInputs(cmd *cobra.Command, args []string) (old, new, oldName, newName string, err error)` (or equivalent exported test hook) in `cmd/drift/input.go`
- [ ] Positional `drift file1 file2` reads both paths with `os.ReadFile`
- [ ] `-` reads `os.Stdin` fully with `io.ReadAll`
- [ ] `--from` / `--to` bypass filesystem and use flag values as content; names `"a/input"` and `"b/input"` for render labels unless you pass empty to `RenderWithNames`
- [ ] `drift - -` produces `old == new` from one stdin read (same bytes for both)
- [ ] `go test ./cmd/drift/...` exits 0
