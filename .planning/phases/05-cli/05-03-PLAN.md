## Objective

Complete `RunE`: call `drift.Diff` and `drift.RenderWithNames` with options derived from flags. Map `--algorithm` strings `myers|patience|histogram` (case-insensitive) to `drift.WithAlgorithm(drift.Myers|Patience|Histogram)`. Map `--context` to `drift.WithContext(n)`; `--no-color` → `drift.WithNoColor()`; `--lang` → `drift.WithLang(v)` when non-empty; `--theme` → `drift.WithTheme(v)` when non-empty; `--split` → `drift.WithSplit()`. Write to `os.Stdout`. Exit **0** when `result.IsEqual`, **1** when differences exist, **2** for usage/IO/render errors — implement via a custom `error` type `exitCodeErr struct{ code int; msg string }` returned from `RunE` that `main` unwraps with `errors.As` (do not call `os.Exit` inside `RunE`).

## must_haves

- [ ] `diffExitCode(1)` / typed error causes `main` to `os.Exit(1)` without printing spurious cobra error text (`SilenceErrors` already true)
- [ ] Identical inputs → exit 0 and **no diff body** (empty hunks — `Render` should write minimal or nothing; match library behavior)
- [ ] All flags from 05-01 affect `Diff`/`Render` option slices
- [ ] Invalid `--algorithm` value returns exit 2 and stderr message containing `algorithm`
- [ ] `go test ./cmd/drift/...` exits 0

**Step 1:** Create `cmd/drift/exit.go`:

```go
type exitCodeErr struct {
  code int
  msg  string
}
func (e *exitCodeErr) Error() string { return e.msg }
```

Add helper `func newExitCode(code int, msg string) error` returning `&exitCodeErr{code, msg}`.

**Step 2:** Implement `parseAlgorithm(s string) (drift.Algorithm, error)` in `main.go` or `flags.go`:

- `strings.ToLower(strings.TrimSpace(s))` switch: `myers` → `drift.Myers`, `patience` → `drift.Patience`, `histogram` → `drift.Histogram`
- default: return error `newExitCode(2, "invalid algorithm: ...")` with substring `invalid algorithm`

**Step 3:** Build `opts []drift.Option` in order: `WithAlgorithm` (if parsed), `WithContext(n)` using flag value (if `n < 0` return exit 2 `invalid context`), `WithNoColor` if flag, `WithLang` if non-empty, `WithTheme` if non-empty, `WithSplit` if flag.

**Step 4:** In `RunE`: call `resolveInputs` → `drift.Diff(old, new, opts...)` — pass **only** diff-relevant options: `WithAlgorithm`, `WithContext` (NOT split/theme/lang/noColor for Diff if those only affect render — **read render.go**: `RenderWithNames` rebuilds config from same `opts ...Option` via `defaultConfig()` loop — so **pass the full opts slice to both `Diff` and `RenderWithNames`**)

**Step 5:** If `Diff` returns error, `return newExitCode(2, err.Error())`.

**Step 6:** If `result.IsEqual`, `return nil` (main exits 0).

**Step 7:** Call `drift.RenderWithNames(result, os.Stdout, oldName, newName, opts...)`; on error `return newExitCode(2, err.Error())`.

**Step 8:** If not equal, `return newExitCode(1, "")` — empty msg so stderr stays clean; document in comment.

**Step 9:** Update `main()`:

```go
err := rootCmd.Execute()
if err != nil {
  var ec *exitCodeErr
  if errors.As(err, &ec) {
    if ec.msg != "" {
      fmt.Fprintln(os.Stderr, ec.msg)
    }
    os.Exit(ec.code)
  }
  fmt.Fprintln(os.Stderr, err)
  os.Exit(2)
}
os.Exit(0)
```
