# Plan 02-03: Wire `WithAlgorithm()` option — integrate all three algorithms into `drift.Diff()`

## Objective

    lines := strings.Split(text, "\n")
    if len(lines) > 0 && lines[len(lines)-1] == "" {

    }
    if len(lines) == 0 {
